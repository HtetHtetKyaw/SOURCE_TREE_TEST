Data Set Up Guide

This guide provides the Data Entry Procedure along with detail order.

Please input data by following order no.


1. Basic Set up Data

Basic data that need to set up first for other set up. Only need to register data that will actually use.

Order No Title Description
1 Address - District District name for Employee address information.
2 Address - Prefecture Prefecture name for Employee address information.
In case of no prefecture, please input same as district.
3 Address - City City name for Employee address information.
4 Address - Township Township name for Employee address information.
5 Bank - Bank Bank name for Employee bank information.
6 Bank - Bank Branch Bank Branch name for Employee bank information.

2. Company related Data

Order No Title Description
7 Company Profile Company policy divided into three part as company general information, weekly off days and salary calculation necessary information.
8 Department Departments of company.
9 Job Title Category of Employee. E.g- HR staff, Web Developer, Android Developer,..
10 Position Detail Position of Employee.E.g - Project Manager, Senior Web Developer...

3. Attendance Related Data

Data which need to set up for daily attendance and attendance calculation.

Order No Title Description
11 Leave Type Kind of leave offered company such as medical leave, earned leave, casual leave,...
12 Work shift Kind of shifts of company
Only need to define when one day payment calculation is depend on working hour in Company Profile set up.
13 Holiday Public Holidays, Company holidays
No need to register weekends.

4. Employee Related Data

Important information of employee.

Order No Title Description
14 Employee All Employee information containing bank information, contract information,...

4. Payroll Related Data

Data need to set up for payroll management
No need to set up when payroll management is not necessary

Order No Title Description
15 Basic Salary Basic Salary of each employee, each department
16 Allowance Allowances offered by company.E.g. transportation fee..
Need to define attendance allowance for overtime calculation
17 Deduction Deductions offered by company.E.g.Social Insurance, Late,...
Need to define attendance when there is deduction for absence and late

5. Optional Data

The following data setups are optional according to company policy.

Order No Title Description Required
18 Income Tax law Tax Law defined by country Required only when tax deduction is need to include in monthly payment caculation.
19 Project Company's project detail information Required only when company want to provide Employee's Project Information and Employee working Info Report.
20 Meeting Room Meeting Room Information of Company Required only when company want to manage meeting room usage.
