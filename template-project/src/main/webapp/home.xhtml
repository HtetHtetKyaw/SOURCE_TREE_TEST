1.3. Users list

# User Group (role) Acronym Roles And Responsibility
1 Admin ADM In charged of all system related Issues including creating users, assign roles, leave types, company profile and fully access to OAS.
2 Manager MGR Supervisor role in OAS simply approve or rejects Attendances and Leaves. Supervisor role can be assigned to any user by the Admin as when needed and allows all access as User role. Moreover they have authorized to view reporting features.
3 User USR Basic Users to request for attendances and leaves.
